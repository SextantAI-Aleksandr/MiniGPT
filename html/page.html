About This Page

This project was created to demonstrate a minimal webpage functionality that vaguely mimicked ChatGPT. Some of the answers from the chat bot will seem insightful, many will seem silly.

About DialoGPT

DialoGPT is an open-source project sponsored by Microsoft to enable "versatile, open-domain chatbots". Check out the project page, the associated GitHub repository, the model hosted on HuggingFace and this technical paper.
